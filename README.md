# AI Data Analyst Agent

An LLM-powered data analysis agent that converts natural language questions into SQL queries, executes them on a DuckDB database, performs data analysis with Pandas, and generates visualizations and analytical reports.

## Features

- Natural language → SQL (DeepSeek)
- DuckDB query execution
- Data analysis with Pandas
- Visualization with Matplotlib
- FastAPI web service

## Tech Stack

- Python
- DeepSeek API
- DuckDB
- Pandas
- Matplotlib
- FastAPI
